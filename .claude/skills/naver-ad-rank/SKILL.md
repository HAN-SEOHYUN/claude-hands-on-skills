# naver-ad-rank — 네이버 파워링크 광고 순위 확인

Python 스크립트로 네이버 검색 결과를 직접 파싱해서 파워링크 광고 순위를 확인한다.
브라우저 불필요. API 키 불필요. Python만 있으면 된다.

---

## 전제 조건

1. **Python 3.8 이상** 설치
2. 스킬 파일 위치: `.claude/skills/naver-ad-rank/fetch_powerlink.py`

---

## 입력 형식

다음 중 하나로 말하면 된다.

```
"[키워드] 검색했을 때 내 광고 순위 알려줘. 내 광고 제목은 [제목]이야."
"[키워드] 파워링크에서 [표시URL] 몇 위야?"
```

**필수 정보 2가지**:
- 검색 키워드 (예: 운동화, 노트북)
- 내 광고 식별 정보 — 둘 중 하나
  - 광고 제목 (예: "무선 비염치료기, 노즈굿")
  - 표시 URL (예: smartstore.naver.com/abc)

## 실행 흐름

### 1. 입력 확인

키워드와 광고 식별 정보가 모두 있는지 확인한다.
빠진 정보가 있으면 아래 안내 문구로 먼저 물어본다.

```
파워링크 광고 순위를 확인하려면 두 가지 정보가 필요합니다.

1. 검색 키워드 — 네이버에서 검색할 단어 (예: 운동화, 비염치료기)
2. 광고 소재 제목 — 내 광고에 표시되는 제목 (예: 무선 비염치료기, 노즈굿)

두 가지를 알려주시면 바로 확인해드립니다!
```

### 2. 스크립트 실행

Bash 툴로 아래 명령어를 실행한다.

```bash
python -X utf8 .claude/skills/naver-ad-rank/fetch_powerlink.py "검색키워드"
```

결과는 JSON 형식으로 반환된다:
```json
{
  "keyword": "가정용비염치료기",
  "ads": [
    {"rank": 1, "title": "광고 제목", "site": "사이트명", "display_url": "example.com/"},
    {"rank": 2, "title": "무선 비염치료기, 노즈굿", "site": "모메조은거", "display_url": "smartstore.naver.com/abc"}
  ]
}
```

### 3. 내 광고 식별

JSON 결과에서 사용자가 제공한 식별 정보로 광고를 찾는다.
오매칭 방지가 최우선 원칙이다. 확실하지 않으면 "찾지 못했다"고 답한다.

#### 우선순위 1 — 표시 URL (사용자가 URL을 제공한 경우)

- 사용자가 제공한 URL의 **도메인 경로 부분**이 광고의 `display_url`에 완전히 포함될 때만 매칭
- 단순 도메인(`naver.com`, `smartstore.naver.com`)만 일치하는 경우는 매칭하지 않는다
- URL 매칭이 성공하면 제목 확인 없이 해당 광고로 확정

#### 우선순위 2 — 광고 제목 (URL 미제공 또는 URL 매칭 실패 시)

**토큰 추출 규칙** (사용자 입력 제목을 분리):
- 공백 기준으로 단어를 분리
- 아래 항목은 매칭 대상 토큰에서 제외:
  - 2글자 이하 단어
  - 숫자 단독 토큰
  - 연도 단독 토큰 (예: "2026", "2025")
  - "NEW", "new", "BEST", "best" 등 마케팅 수식어

**매칭 조건**:
- 추출된 의미 토큰이 **모두** 광고 제목(title)에 포함되어야 매칭
- 하나라도 빠지면 불일치로 처리
- 대소문자 구분 없이 비교

### 4. 결과 파일 저장

내 광고 식별이 끝나면 `fetch_powerlink.py`를 `--save-dir`과 `--my-rank` 옵션으로 재실행한다.
`MY_RANK`에는 식별된 순위(정수), 못 찾은 경우 `None`을 넣는다.

```bash
python -X utf8 .claude/skills/naver-ad-rank/fetch_powerlink.py "검색키워드" --save-dir ".claude/skills/naver-ad-rank/data" --my-rank MY_RANK
```

내 광고를 찾지 못한 경우:

```bash
python -X utf8 .claude/skills/naver-ad-rank/fetch_powerlink.py "검색키워드" --save-dir ".claude/skills/naver-ad-rank/data"
```

반환된 JSON의 `saved_path` 값을 `saved_path`로 기억한다.

### 5. 결과 출력

아래 형식으로 출력한다. 저장 경로는 4단계에서 얻은 `saved_path`를 그대로 출력한다.

```
🔍 "가정용비염치료기" 파워링크 검색 결과

순위  광고 제목                         표시 URL
──────────────────────────────────────────────────
1위   비염 공기청정기는 따로있어요       koenjam.kr/
2위 ★ 무선 비염치료기, 노즈굿           smartstore.naver.com/...   ← 내 광고
3위   ...

📌 내 광고: 2위 / 전체 8개 중

💾 결과 저장됨: .claude/skills/naver-ad-rank/data/가정용비염치료기_20260528_1430.txt
```

내 광고가 없으면:

```
📌 "가정용비염치료기" 검색 결과에서 내 광고를 찾지 못했습니다.
   확인한 광고 식별 정보: 무선 비염치료기, 노즈굿
   전체 파워링크 8개를 확인했습니다.

💾 결과 저장됨: .claude/skills/naver-ad-rank/data/가정용비염치료기_20260528_1430.txt
```

#### 저장 실패 시

4단계 스크립트가 오류를 반환하면 아래 안내만 추가하고 결과 표시는 계속한다.

```
⚠️ 파일 저장에 실패했습니다. 결과는 채팅창에서 확인하세요.
```

## 복수 키워드 일괄 조회

여러 키워드를 한꺼번에 확인하려면 목록을 전달한다.

```
키워드 3개 순위 확인해줘.
- 비염치료기
- 가정용비염치료기
- 휴대용비염치료기
내 광고 제목은 "무선 비염치료기, 노즈굿"이야.
```

각 키워드마다 스크립트를 순서대로 실행하고, 결과를 표로 합산 출력한다.

```
📊 키워드별 파워링크 순위 요약

키워드             내 광고 순위  전체 광고 수
────────────────────────────────────────
비염치료기         2위           10개
가정용비염치료기   순위 없음     8개
휴대용비염치료기   1위           6개
```
